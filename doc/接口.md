# 接口文档
## 登陆
### url
`/login`
### 参数
   参数名  |    类型    |   约束       
:----- :  |   :---:   | :----------  
username  |   字符串   | 5-16个字符    
password  |   字符串   | 密码md5(128位)

### 结果
```json
{
    "result":"success",
    "data":{
        "token":"ABCDEFG"
    }
}
```

```json
{
    "result":"error",
    "data":{
        "err_code": "001"
    }
}
```
错误码请参考 “错误码.md”
